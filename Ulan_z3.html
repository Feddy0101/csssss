Nowe możliwości CSS w porównaniu z HTML:

  1. Selektory zaawansowane - możliwość stylowania na podstawie atrybutów, struktury DOM itd.
  2. Animacje i przejścia - tworzenie efektów wizualnych bez użycia JavaScriptu.
  3. Układ strony (layout) - np. Grid, Flexbox, które umożliwiają nowoczesne i elastyczne rozmieszczanie elementów.
  4. Zmienne CSS - możliwość tworzenia dynamicznych, powtarzalnych wartości
  5. Zagnieżdżanie stylów (w CSS preprocesorach) - lepsza organizacja kodu (np. SASS, LESS).

Wady css

  • Brak pełnej izolacji stylów - style mogą się niechcący nakładać (problem z tzw. "specificity").
  • Trudności w debugowaniu - szczególnie przy złożonych strukturach stylów.
  • Zależność od struktury HTML - zmiana w HTML może wpłynąć na działanie stylu.
  • Problemy z kompatybilnością przeglądarek - niektóre właściwości CSS mogą nie działać jednakowo wszędzie.
  • Brak kontroli logiki - CSS służy wyłącznie do prezentacji, nie do działania (jak np. JavaScript)
