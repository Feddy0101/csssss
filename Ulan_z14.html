Sposoby definiowania (określania) kolorów
Sposób Opis Przykład
RGB Określanie z użyciem definicji RGB color: rgb(132, 187, 251);
HEX Określanie z użyciem definicji heksadecymalnych color: purple;
Nazwa koloru Określanie z użyciem nazwy kolorów div { color:blue; }
RGB + nasycenie koloru Określanie z użyciem definicji RGB oraz nasycenia kolorów div { color: rgb(41%, 62%, 78%); }
